# REACT

- Reużywalne moduły
		- HTML + JSX + CSS

- Konwencja mówi że każdy index.html Reactowy powinien zawierać `div` o `id="root"`

## JSX
- Umożliwia używanie syntaxu html wewnątrz JS, np:
		-	<h1> hello my name is {name} </h1>
